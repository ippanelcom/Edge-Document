With these endpoints, you can manage your documents, including adding, updating, deleting, and listing documents.

## Base URL

All user document management endpoints use the base URL: `https://edge.ippanel.com/v1`

## Authentication

All endpoints in this section require authentication. You must include your authorization token in the request header:

```
Authorization: YOUR_TOKEN_HERE
```

## Available User Documents Endpoints

The IPPanel Edge API offers endpoints for managing your documents:

- [List Documents](./list-document) - List all documents.
- [Upload Document](./upload-document) - upload document.
- [Remove Document](./remove-document) - remove document.
